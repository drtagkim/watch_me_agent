# 👁️ Watch Me Agent (v0.0.3)

**Watch Me Agent**는 발표자의 웹캠 영상(Multi-frame GIF)과 실시간 오디오를 융합하여, **Gemini 3.1 Pro Preview**의 막강한 멀티모달 능력을 통해 발표 내용, 비언어적 맥락, 핵심 질문 사항 등을 리얼타임으로 심층 분석하는 AI 조교 시스템입니다. 

개발자: **김태경 & 윤이나**

---

## 💻 실행 및 권장 환경 (Requirements)
- **최적화 기기:** Apple Silicon (MacBook Pro M3 기준 최적화 및 동작 확인 완료)
- **하드웨어 제어:** Mac 내장 카메라 및 마이크(자동 검색 및 권한 획득)
- **파이썬 환경:** **가상 환경(`venv`) 활성화 필수**

## 🛠 주요 핵심 기능
1. **Continuous 3-Second GIF Capture:** 단순한 1장의 정지 화면이 아닌, 3초 동안의 다중 프레임 움직임을 캡처하여 애니메이션(GIF) 형태로 병합 및 Gemini에게 전송합니다.
2. **Micro/Macro Questioning:** 화자의 STT(오디오 스크립트)와 영상의 맥락을 교차 파악하여 구체적/지엽적 허점을 찌르는 질문(Micro)과, 비즈니스/철학적 관점의 큰 그림 질문(Macro)을 실시간 도출합니다.
3. **Dynamic Context Formatting:** `--focus` 옵션을 통해 심사 분야(예: 투자 피칭, 학술 심사 등)를 설정하면, Gemini 모델 스스로 해당 목적에 최적화된 맞춤형 분석 지침을 도출합니다.
4. **Volume Control:** Numpy 연산을 거친 1차 무음 필터링과 판단 모듈에서의 2차 잡음 필터링으로 낭비성 API 콜을 선제적으로 차단합니다.

## 🏃 실행 방법
해당 툴의 주 목적은 백그라운드 구동이며, 반드시 **가상환경(`venv`)**이 적용된 파이썬 환경에서 실행해야 합니다. 다음 두 가지 방법 중 편한 방식을 선택하세요.

### 방법 1 (권장): 원클릭 자동 실행 스크립트
자동으로 가상환경을 `activate`하고, `.env` 파일을 로드하며, 상호작용으로 분석 옵션을 설정할 수 있습니다.
```bash
./watch_run.sh
```

### 방법 2: 수동(CLI) 파라미터 실행
터미널에서 직접 가상환경을 켜고, 매개변수를 넘겨 실행합니다.
```bash
source venv/bin/activate
export GEMINI_API_KEY="본인의_API_키"  # .env 파일이 없다면 입력
python watch_and_analyze.py --chunk 10 --focus "투자 피칭"
```

## ⌨️ 단축키 및 종료 방법
분석은 백그라운드 스레드에서 끊기지 않고 계속 돕니다. 프로그램과 기록을 마치고 싶다면 메인 터미널 창에서 다음 중 하나를 실행하세요:
- **`[Enter]`** : 키보드 엔터 키를 누르면 즉시 캡처를 멈추고 분석을 종료합니다.
- **`[Ctrl + C]`** : 강제 인터럽트를 발생시켜 안전하게 프로세스를 중단시킵니다.
> **💡 윤이나의 팁:** 위 단축키를 눌러 종료 신호를 보내면 그 즉시 강제로 꺼지는 것이 아닙니다. *현재까지 밀려있는 백그라운드 분석을 모두 끝마친 후*, 모인 데이터를 싹~ 긁어모아 **최종 요약본(Final Summary)**을 생성하고 안전하게 하드웨어를 반납한 뒤 우아하게 종료됩니다!
